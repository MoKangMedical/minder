# 念念 (Minder)

你的第二记忆，珍藏每一个念想

## 🌐 在线访问

GitHub Pages: https://mokangmedical.github.io/minder/

## 📱 关于

念念是一款基于AI的智能语音提醒应用，让你通过自然语言说出念想，AI自动理解整理，在合适的时候温柔提醒。

## ✨ 功能

- 🎤 语音输入 - 说出你的念想
- 🤖 AI理解 - 自动提取时间、地点、分类
- 💾 本地存储 - 数据保存在本地
- 📱 PWA支持 - 可以添加到主屏幕
- 💝 念想卡片 - 完成后获得鼓励

## 🛠️ 技术

- HTML5 / CSS3 / JavaScript
- Web Speech API
- Kimi AI API
- LocalStorage
- Service Worker
