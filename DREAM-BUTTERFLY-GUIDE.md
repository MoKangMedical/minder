## 🛠️ 技术实现建议

### 方案A：使用视频（高质量）
1. 在即梦生成MP4视频
2. 转换为WebM格式（更小体积）
3. 使用HTML5 video标签嵌入
4. 添加poster属性作为封面

```html
<video autoplay loop muted playsinline poster="butterfly-poster.jpg">
    <source src="butterfly.webm" type="video/webm">
    <source src="butterfly.mp4" type="video/mp4">
</video>
```

### 方案B：使用GIF（简单）
1. 将视频转换为GIF
2. 使用img标签
3. 注意控制文件大小（< 500KB）

### 方案C：使用Lottie（最佳性能）
1. 在即梦生成后，使用After Effects导出
2. 使用Bodymovin插件导出为JSON
3. 使用Lottie-web库播放
4. 矢量动画，无限清晰

```html
<div id="butterfly-animation"></div>
<script>
    lottie.loadAnimation({
        container: document.getElementById('butterfly-animation'),
        renderer: 'svg',
        loop: true,
        autoplay: true,
        path: 'butterfly.json'
    });
</script>
```

### 方案D：CSS动画（当前实现）
- 无需外部资源
- 性能最好
- 完全可控
- 已为你实现 ✓
