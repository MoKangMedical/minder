# 📱 念念小红书自动发布脚本 - 使用指南

## 🚀 快速开始 (5分钟完成)

### 第一步：下载 AutoX.js APP

**下载地址** (选1个)：
1. **GitHub**: https://github.com/kkevsekk1/AutoX/releases
2. **国内镜像**: https://autoxoss.autoxingshu.com/
3. **应用商店**: 搜索 "AutoX" 或 "Auto.js"

**下载**: `autox-v6.x.x-arm64-v8a.apk` (根据手机选择版本)

---

### 第二步：安装并设置

1. **安装 APK**
   ```bash
   # 允许安装未知来源应用
   设置 → 安全 → 允许未知来源 → 开启
   ```

2. **开启无障碍权限** (必须！)
   ```
   设置 → 无障碍 → AutoX.js → 开启
   ```
   
   或脚本会自动引导你开启

3. **开启悬浮窗权限** (可选但推荐)
   ```
   设置 → 应用管理 → AutoX.js → 悬浮窗 → 允许
   ```

---

### 第三步：导入脚本

#### 方法1: 直接复制 (推荐)

1. 打开 AutoX.js APP
2. 点击右下角 **"+"** → **"文件"**
3. 文件名: `xiaohongshu_auto_publish.js`
4. 复制粘贴脚本内容 (从 `/root/.openclaw/workspace/xiaohongshu_auto_publish.js`)
5. 点击保存

#### 方法2: 文件导入

1. 将脚本文件传到手机
2. AutoX.js → 菜单 → 导入 → 选择文件

---

### 第四步：准备图片

**在小红书发布前，你需要准备5张图片**：

1. **封面图** (最重要！)
   - 念念APP启动界面
   - 加手机外壳
   - 标题："你的第二记忆"

2. **痛点图**
   - 文字：忘事的场景
   - 引发共鸣

3. **功能图**
   - 录音界面截图
   - 展示核心功能

4. **效果图**
   - AI解析结果
   - 念想卡片

5. **转化图**
   - 价格对比
   - 二维码

**图片位置**: 保存到手机相册，方便选择

---

### 第五步：运行脚本

1. **打开脚本**
   - AutoX.js 中找到 `xiaohongshu_auto_publish.js`
   - 点击运行

2. **填写内容** (界面会显示)
   - 标题：已预填好
   - 正文：已预填好
   - 标签：已预填好
   - 可根据需要修改

3. **点击"开始自动发布"**

4. **脚本会自动**：
   - ✅ 打开小红书APP
   - ✅ 点击发布按钮
   - ✅ 进入图文模式
   - ✅ 提示你选择图片
   - ✅ 填写标题和正文
   - ✅ 点击发布

---

```
解决：检查无障碍权限是否开启
```

**问题2: 找不到元素**
```
解决：小红书界面更新，需要调整脚本
```

**问题3: 图片选择失败**
```
解决：需要手动选择图片，脚本会暂停等待
```

## 🆘 遇到问题？

### 常见错误：

**错误1**: `请开启无障碍服务`
```
解决: 设置 → 无障碍 → 找到AutoX.js → 开启
```

**错误2**: `无法找到小红书APP`
```
解决: 确保小红书已安装，或修改包名
```

**错误3**: `点击无反应`
```
解决: 重新开启无障碍权限，重启APP
```
